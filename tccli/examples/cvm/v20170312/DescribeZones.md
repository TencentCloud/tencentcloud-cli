**Example 1: 查询可用区信息（可用区中文名称）**

查询广州地域的可用区信息。返回可用区的中文名称需要传参Language=zh-CN。

Input: 

```
tccli cvm DescribeZones --cli-unfold-argument ```

        "TotalCount": 5,
        "ZoneSet": [

            {
                "ZoneState": "AVAILABLE",
