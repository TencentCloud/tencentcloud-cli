**Example 1: 错误示例**

人员ID不存在

Input: 

```
tccli iai DeleteFace --cli-unfold-argument  \
    --PersonId 5001 \
    --FaceIds 2875186538564559728
```

Output: 
```
{
    "Response": {
        "Error": {
            "Code": "ResourceNotFound.ErrorPersonNotExisted",
            "Message": "个体不存在。"
        },
        "RequestId": "e2b603e5-51d0-4a8a-8319-2df026ae7518"
    }
}
```
