**Example 1: 错误示例**

人员ID不存在

Input: 

```
tccli iai DeletePerson --cli-unfold-argument  \
    --PersonId 3001
```

Output: 
```
{
    "Response": {
        "Error": {
            "Code": "ResourceNotFound.ErrorPersonNotExisted",
            "Message": "个体不存在。"
        },
        "RequestId": "4b9c1bdc-86eb-413f-8c74-15bcf4466cd5"
    }
}
```
