**Example 1: 获取独享集群大版本镜像列表**

本接口用于获取独享集群大版本镜像列表

Input: 

```
tccli dlc DescribeDataEngineImageVersions --cli-unfold-argument  \
    --EngineType SQL \
    --Sort InsertTime \
    --Asc False
```
