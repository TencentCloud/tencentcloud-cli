**Example 1: 创建一个CCN**

创建一个云联网CCN实例

Input: 

```
tccli vpc CreateCcn --cli-unfold-argument  \
    --CcnDescription test+description \
    --Tags.0.Value shanghai \
    --Tags.0.Key city \
    --CcnName test+name \
    --QosLevel PT
```
