**Example 1: 关闭传统弹性公网IPv6地址带宽**

入参传递Ip6AddressIds。

Input: 

```
tccli vpc ReleaseIp6AddressesBandwidth --cli-unfold-argument  \
    --Ip6AddressIds eip-31g37kea
```

Output: 
```
{
    "Response": {
        "TaskId": "12579789",
        "RequestId": "fe6aa94a-2a74-41d8-9570-337a575443fe"
    }
}
```
