**Example 1: 创建IP地址模板**

创建IP地址模板

Input: 

```
tccli vpc CreateAddressTemplate --cli-unfold-argument  \
    --AddressTemplateName TestName \
    --Addresses 192.168.1.0/24 192.168.2.0/24
```
