**Example 1: 查询IP地址模板**

查询IP地址模板

Input: 

```
tccli vpc DescribeAddressTemplates --cli-unfold-argument  \
    --Filters.0.Name address-template-name \
    --Filters.0.Values demo \
    --Filters.1.Name address-template-id \
    --Filters.1.Values ipm-mdunqeb6
```
