**Example 1: 删除指定版本的凭据**



Input: 

```
tccli ssm DeleteSecretVersion --cli-unfold-argument  \
    --SecretName lzctest \
    --VersionId v1.0
```

Output: 
```
{
    "Response": {
        "RequestId": "c701cf6d-4199-4822-8630-d48ee953f485",
        "SecretName": "lzctest",
        "VersionId": "v1.0"
    }
}
```
