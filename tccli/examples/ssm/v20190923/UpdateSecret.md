Input: 

```
tccli ssm UpdateSecret --cli-unfold-argument  \
    --SecretName lzctestss \
    --VersionId v1.0 \
    --SecretString 'new value'
```

```
{
    "Response": {

        "SecretName": "lzctestss",
        "VersionId": "v2.0"
```
