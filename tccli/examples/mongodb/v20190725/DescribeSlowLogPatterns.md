**Example 1: 查询数据库慢日志统计信息**



Input: 

```
tccli mongodb DescribeSlowLogPatterns --cli-unfold-argument  \
    --InstanceId cmgo-a3bm**** \
    --StartTime '2020-04-20 00:00:00' \
    --EndTime '2020-04-20 22:00:00' \
    --SlowMS 100 \
    --Offset 0 \
    --Limit 100
```
