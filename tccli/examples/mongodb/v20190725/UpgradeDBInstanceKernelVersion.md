**Example 1: 升级实例当前内核版本号**

当新版本内核被推出时，可以通过升级内核版本到最新版本

Input: 

```
tccli mongodb UpgradeDBInstanceKernelVersion --cli-unfold-argument  \
    --InstanceId cmgo-p8vnipr5
```

Output: 
```
{
    "Response": {
        "RequestId": "2efcf780-9970-11ec-8975-6bc44e3bfad7",
        "FlowId": 19677
    }
}
```
