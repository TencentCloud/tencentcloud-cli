**Example 1: 获取下载程序包信息**

获取应用application-zxxxxxxx下程序包id为pkg-xxxxxxx的下载信息

Input: 

```
tccli tsf DescribeDownloadInfo --cli-unfold-argument  \
    --ApplicationId application-zxxxxxxx \
    --PkgId pkg-xxxxxxx
```
