**Example 1: 取消资产忽略项**

取消资产忽略项

Input: 

```
tccli tcss DeleteComplianceAssetPolicySetFromWhitelist --cli-unfold-argument  \
    --AssetItemId 2202462 \
    --CustomerPolicyItemIdSet 2809 \
    --AssetType ASSET_CONTAINER
```

Output: 
```
{
    "Response": {
        "RequestId": "925e6419-1fb8-4169-a96c-020e3ae9171e"
    }
}
```
