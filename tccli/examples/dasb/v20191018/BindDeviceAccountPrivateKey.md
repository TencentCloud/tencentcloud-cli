**Example 1: 绑定主机账号私钥**

绑定主机账号私钥

Input: 

```
tccli dasb BindDeviceAccountPrivateKey --cli-unfold-argument  \
    --PrivateKey sfefdfsfewsgf \
    --Id 1 \
    --PrivateKeyPassword 4wegdffdew
```

Output: 
```
{
    "Response": {
        "RequestId": "b8961a60-cee7-46b0-a993-2dd233f413fc"
    }
}
```
