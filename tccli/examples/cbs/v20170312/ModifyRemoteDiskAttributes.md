**Example 1: 修改单副本SSD硬盘名称**

修改一个单副本SSD硬盘的名称。

Input: 

```
tccli cbs ModifyRemoteDiskAttributes --cli-unfold-argument ```

Output: 
```
{
    "Response": {
        "RequestId": "a0ed56a0-4b9a-4690-bd5e-7aa76fd67b4d"
    }
}
```

**Example 2: 批量修改单副本SSD硬盘项目ID**

批量修改多个单副本SSD硬盘的项目ID。

Input: 

```
tccli cbs ModifyRemoteDiskAttributes --cli-unfold-argument ```

Output: 
```
{
    "Response": {
        "RequestId": "a0ed56a0-4b9a-4690-bd5e-7aa76fd67b4d"
