**Example 1: 创建普通分账单元**



Input: 

```
tccli billing CreateAllocationUnit --cli-unfold-argument  \
    --Name 财务部 \
    --ParentId 1
```

Output: 
```
{
    "Response": {
        "Id": 2,
        "RequestId": "5ac3b178-3434-452e-9a1e-dc8b8089a2d8"
    }
}
```
