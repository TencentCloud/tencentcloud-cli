**Example 1: 修改计划名称**

 

Input: 

```
tccli iss UpdateRecordPlan --cli-unfold-argument  \
    --PlanId 88ac5ea6c1f**********24671d0f94f \
    --Mod.PlanName newname
```

```
{
    "Response": {
        "Data": {
            "PlanId": "88ac5ea6c1f**********24671d0f94f",

            "RepairMode": 0,
```
