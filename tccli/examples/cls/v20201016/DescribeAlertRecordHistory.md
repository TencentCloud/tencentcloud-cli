**Example 1: 查询未恢复告警**

查询指定时间段内尚未恢复的告警

Input: 

```
tccli cls DescribeAlertRecordHistory --cli-unfold-argument  \
    --Offset 0 \
    --Limit 20 \
    --From 1681401600000 \
    --To 1681487999999 \
    --Filters.0.Key status \
    --Filters.0.Values 0
```
