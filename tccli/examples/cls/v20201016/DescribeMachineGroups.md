**Example 1: 获取机器组列表**

获取包含指定标签的机器组列表

Input: 

```
tccli cls DescribeMachineGroups --cli-unfold-argument  \
    --Filters.0.Key tagKey \
    --Filters.0.Values k1 \
    --Offset 0 \
    --Limit 10
```
