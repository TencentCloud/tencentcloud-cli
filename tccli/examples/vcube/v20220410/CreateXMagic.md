**Example 1: 开通优图美视**

需要优图美视功能的用户，可以开通优图美视功能

Input: 

```
tccli vcube CreateXMagic --cli-unfold-argument  \
    --ApplicationId 1 \
    --CompanyPermit abc \
    --CompanyType abc \
    --CompanyName abc \
    --XMagicResourceIds abc
```

Output: 
```
{
    "Response": {
        "RequestId": "abc"
    }
}
```
