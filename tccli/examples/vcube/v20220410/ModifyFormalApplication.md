**Example 1: 修改正式应用包名**

修改正式应用AppName，国际站专有

Input: 

```
tccli vcube ModifyFormalApplication --cli-unfold-argument  \
    --AppName nengc \
    --BundleId com.nengc.dd \
    --PackageName com.nengc.dd \
    --ApplicationId 234423
```

Output: 
```
{
    "Response": {
        "RequestId": "3c140219-cfe9-470e-b241-907877d6fb03"
    }
}
```
