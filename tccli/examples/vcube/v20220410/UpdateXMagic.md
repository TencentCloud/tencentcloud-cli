**Example 1: 更新xmagic**

更新xmagic申请信息，  Status 为2，3的时候可用

Input: 

```
tccli vcube UpdateXMagic --cli-unfold-argument  \
    --XMagicResourceId abc \
    --CompanyPermit abc \
    --CompanyType abc \
    --CompanyName abc \
    --XMagicId 1
```

Output: 
```
{
    "Response": {
        "RequestId": "abc"
    }
}
```
