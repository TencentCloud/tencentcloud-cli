**Example 1: 创建安卓应用**

创建安卓应用

Input: 

```
tccli gs CreateAndroidApp --cli-unfold-argument  \
    --Name 测试App1 \
    --UserId user1 \
    --AppMode NORMAL
```

```
{
    "Response": {
        "AndroidAppId": "apk-f5nqwdvg",
        "RequestId": "3b675310-8ec8-46c0-ae50-dc2944667c6f"
    }
}
```
