**Example 1: 私有域-添加A记录**

添加A记录

Input: 

```
tccli privatedns CreatePrivateZoneRecord --cli-unfold-argument  \
    --ZoneId zone-123456 \
    --RecordType A \
    --SubDomain b \
    --RecordValue 3.3.3.3 \
    --Weight 100 \
    --TTL 600
```

Output: 
```
{
    "Response": {
        "RequestId": "a98891db-9d73-514a-8751422197b540cd",
        "RecordId": "15421"
    }
}
```
