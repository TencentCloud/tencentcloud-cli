**Example 1: 获取私有域解析账号列表**



Input: 

```
tccli privatedns DescribePrivateDNSAccountList --cli-unfold-argument  \
    --Limit 200 \
    --Offset 0 \
    --Filters.0.Name AccountUin \
    --Filters.0.Values 1000230100
```
