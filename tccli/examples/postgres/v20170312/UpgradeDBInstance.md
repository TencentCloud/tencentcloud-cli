**Example 1: 升级实例**

调整实例postgres-6bwgamo3规格为内存2GB、存储100GB。

Input: 

```
tccli postgres UpgradeDBInstance --cli-unfold-argument  \
    --Storage 200 \
    --DBInstanceId postgres-6bwgamo3 \
    --Memory 2
```
