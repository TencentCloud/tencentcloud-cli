**Example 1: 获取某代理商的解绑客户列表**



Input: 

```
tccli partners DescribeUnbindClientList --cli-unfold-argument  \
    --Status 1 \
    --UnbindUin 100002 \
    --ApplyTimeEnd 2020-09-22 \
    --Limit 1 \
    --Offset 1 \
    --ApplyTimeStart 2020-09-22
```

        "TotalCount": 1,

    }
}
