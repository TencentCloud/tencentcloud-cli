**Example 1: 创建搜索下载攻击日志任务**

创建搜索下载攻击日志任务

Input: 

```
tccli waf PostAttackDownloadTask --cli-unfold-argument  \
    --Domain waf.com \
    --StartTime 2020-09-22T00:00:00+00:00 \
    --EndTime 2020-09-22T00:00:00+00:00 \
    --QueryString bot:1 \
    --TaskName task-name \
    --Sort desc \
    --Count 0
```
