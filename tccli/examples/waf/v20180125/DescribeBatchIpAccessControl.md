**Example 1: Waf 多域名ip黑白名单查询**



Input: 

```
tccli waf DescribeBatchIpAccessControl --cli-unfold-argument  \
    --Sort ts_version:-1 \
    --Limit 20 \
    --Filters.0.Values Domain \
    --Filters.0.Name www.q.com \
    --Filters.0.ExactMatch True \
    --OffSet 0
```
