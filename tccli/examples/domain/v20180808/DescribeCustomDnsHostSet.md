**Example 1: 查询自定义DNS Host列表**



Input: 

```
tccli domain DescribeCustomDnsHostSet --cli-unfold-argument  \
    --DomainId domain-esysdasdgq \
    --Offset 0 \
    --Limit 3
```

                    "8.8.9.9",
