**Example 1: 修改媒体名称**

 

Input: 

```
tccli cme ModifyMaterial --cli-unfold-argument  \
    --Platform 1000000009 \
    --MaterialId 5fd8ad3d628dc30001bd0895 \
    --Name 修改媒体名称
```

Output: 
```
{
    "Response": {
        "RequestId": "7bf08361-4455-4cbd-afd6-423b62c54a05"
    }
}
```
