**Example 1: 分页获取视频导出任务列表**

 

Input: 

```
tccli cme DescribeTasks --cli-unfold-argument  \
    --Platform 1000000009 \
    --Offset 0 \
    --Limit 10
```

        "TotalCount": 1,
        "TaskBaseInfoSet": [
            {
                "CreateTime": "2020-04-30T03:26:37Z",

                "Status": "SUCCESS",
