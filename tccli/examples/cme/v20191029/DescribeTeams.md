**Example 1: 分布获取团队信息**



Input: 

```
tccli cme DescribeTeams --cli-unfold-argument  \
    --Platform 1000000009 \
    --Limit 10 \
    --Offset 0
```

```
{
    "Response": {
        "TotalCount": 1,
        "TeamSet": [
            {
                "TeamId": "1kdk882ddd88338ddd3k3",
                "Name": "工作室1",
                "MemberCount": 30,
                "CreateTime": "2020-03-09T02:35:21Z",
                "UpdateTime": "2020-07-06T04:55:21Z"
            }
        ],
        "RequestId": "98534d9f-aeec-4817-bbe2-d3499f95a2af"
    }
}
```
