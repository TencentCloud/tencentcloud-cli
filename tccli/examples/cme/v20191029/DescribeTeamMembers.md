**Example 1: 拉取指定成员的信息**



Input: 

```
tccli cme DescribeTeamMembers --cli-unfold-argument  \
    --Platform 1000000009 \
    --TeamId 1kdk882ddd88338ddd3k3 \
    --MemberIds 1993939392iidkei8ei
```

Output: 
```
{
    "Response": {
        "TotalCount": 1,
        "MemberSet": [
            {
                "MemberId": "1993939392iidkei8ei",
                "Remark": "主编辑李四",
                "Role": "Members"
            }
        ],
        "RequestId": "99004d9f-aeec-4817-bbe2-d3499f95a2bf"
    }
}
```

