**Example 1: 查询项目列表示例**

当用户在自己的业务系统中想进行产品管理时，可以通过查询项目列表接口查询用户在物联网开发平台创建的项目。

Input: 

```
tccli iotexplorer GetProjectList --cli-unfold-argument  \
    --Offset 0 \
    --Limit 0
```
