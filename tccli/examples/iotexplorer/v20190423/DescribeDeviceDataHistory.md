**Example 1: 获取设备历史数据**

获取设备历史数据

Input: 

```
tccli iotexplorer DescribeDeviceDataHistory --cli-unfold-argument  \
    --MinTime 1548677099000 \
    --MaxTime 1548763499000 \
    --ProductId LJ0INDNU7U \
    --DeviceName light1 \
    --FieldName color \
    --Limit 5
```
