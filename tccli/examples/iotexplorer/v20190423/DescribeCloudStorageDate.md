**Example 1: 获取具有云存的日期**



Input: 

```
tccli iotexplorer DescribeCloudStorageDate --cli-unfold-argument  \
    --ProductId AQTV2839QJ \
    --DeviceName p2p_32731213_1
```

Output: 
```
{
    "Response": {
        "RequestId": "77d90d07-e48c-4cf0-bec5-afe5b137f80e",
        "Data": [
            "2021-01-14"
        ]
    }
}
```
