Input: 

```
tccli cdb DescribeCPUExpandStrategyInfo --cli-unfold-argument  \
    --InstanceId cdb-sad1dsfa
```

```
{
    "Response": {
        "Type": "auto",
        "ExpandCpu": "4",
        "AutoStrategy": {
            "ExpandThreshold": 0,
            "ShrinkThreshold": 0,
            "ExpandPeriod": 0,
            "ShrinkPeriod": 0
        },
        "RequestId": "sasd8asf-f78asd-fas8-dga9-gdfadf1234"
    }
}
```
