**Example 1: 修改数据库备份配置**

修改数据库备份配置

Input: 

```
tccli cdb ModifyBackupConfig --cli-unfold-argument  \
    --InstanceId cdb-c1nl9rpv \
    --ExpireDays 10 \
    --BinlogExpireDays 8 \
    --BackupMethod physical \
    --StartTime 02:00-06:00
```

{
    "Response": {
