**Example 1: 查询加固实例列表**

根据加固实例的包名和名称进行筛选，并返回最新的20条记录

```
tccli ms DescribeShieldInstances --cli-unfold-argument  \
    --Limit 20 \
    --Filters.0.Name AppPkgName \
    --Filters.0.Value com.tencent.mm \
    --Filters.1.Name AppName \
    --Filters.1.Value wechat \
    --Offset 0
```
