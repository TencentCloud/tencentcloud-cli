**Example 1: 删除MCP Server**

删除指定实例下的MCP Server。

Input: 

```
tccli lighthouse RemoveMcpServers --cli-unfold-argument  \
    --InstanceId lhins-5h3s0lht \
    --McpServerIds lhms-p2dophob
```

Output: 
```
{
    "Response": {
        "RequestId": "a50de509-5ecb-40db-9ad2-a01cffc8c3f3"
    }
}
```
