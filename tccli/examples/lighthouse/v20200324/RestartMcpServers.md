**Example 1: 重启MCP Server**

重启指定实例下的MCP Server。

Input: 

```
tccli lighthouse RestartMcpServers --cli-unfold-argument  \
    --InstanceId lhins-5h3s0lht \
    --McpServerIds lhms-p2dophob
```

Output: 
```
{
    "Response": {
        "RequestId": "dfca933c-ba03-4c91-a03b-db81f79e0419"
    }
}
```
