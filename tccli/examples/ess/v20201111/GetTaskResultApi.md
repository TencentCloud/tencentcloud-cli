通过转换任务Id查询任务状态

Input: 

```
tccli ess GetTaskResultApi --cli-unfold-argument  \
    --Operator.UserId yDxoQxxxxxxxxxxxxxxxxxx1h2hnnR \
    --TaskId 2022xxxxxxxxxx992850
```

```
{
    "Response": {

        "TaskId": "2022072xxxx992850",
```
