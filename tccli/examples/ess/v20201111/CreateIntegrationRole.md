**Example 1: 示例-主代子创建角色**

主企业代理子企业创建角色，设置 Agent 参数，并设置 ProxyOrganizationId 为子企业id

Input: 

```
tccli ess CreateIntegrationRole --cli-unfold-argument  \
    --Name 业务员角色 \
    --Description 这是角色描述信息 \
    --IsGroupRole 0 \
    --Operator.UserId y******************5 \
    --Agent.ProxyOrganizationId y**********************J
```

Output: 
```
{
    "Response": {
        "RoleId": "abc",
        "RequestId": "abc"
    }
}
```
