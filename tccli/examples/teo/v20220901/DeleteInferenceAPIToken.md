**Example 1: 删除推理 API Token**

删除指定 ID 的推理 API Token。

Input: 

```
tccli teo DeleteInferenceAPIToken --cli-unfold-argument  \
    --ZoneId zone-2***8ev000 \
    --TokenId deb88f89-92b***3f3466bbb59b
```

Output: 
```
{
    "Response": {
        "RequestId": "52b48b4f-1145-4b77-845e-1e8975cac7f4"
    }
}
```
