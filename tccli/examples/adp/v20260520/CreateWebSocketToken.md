**Example 1: 创建WS的token**



Input: 

```
tccli adp CreateWebSocketToken --cli-unfold-argument  \
    --Type 2 \
    --AppId 2051960827559960192
```

Output: 
```
{
    "Response": {
        "AppId": "2051960827559960192",
        "Token": "44ad7d4f-bad8-4220-9a44-318b76d7f1f8",
        "RequestId": "2b962a9d-8c18-45af-8530-a937a518c500"
    }
}
```
